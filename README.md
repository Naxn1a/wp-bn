# How to use

Just clone this project:
```bash
git clone https://github.com/Naxn1a/workspace-bn.git
```
## Local setup
### Install the dependencies
```bash
npm install
```

### Setup `.env` file

```bash
CLIENT={client_url}
JWT={secret}
DATABASE_URL={database_url}
```

### `Develop`
```bash
npm run dev
# or
yarn dev
```

### `Start`
```bash
npm start
# or
yarn start
```

### Goodluck!
